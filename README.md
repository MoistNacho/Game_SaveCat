# MiniGame - Save Cat
> 웹기반 HTML, CSS, Vanilla JS를 모두 활용해 만든 미니게임 입니다.

Demo : https://moistnacho.github.io/Game_SaveCat/

## 게임방법
> 제한시간 안에 모든 고양이를 클릭해서 구해주면 됩니다. 

## Tools
+ Vanilla JS
+ Chrome Browser
